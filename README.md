## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/bcExpt1123/java-alda.git
    ```
2. Navigate to the project directory:
    ```sh
    cd java-alda
    ```

## Usage

Run code:
```sh
  mvn clean compile exec:java
```
Run test:
```sh
  mvn test
```
